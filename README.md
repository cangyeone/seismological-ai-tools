# 地震学AI工具

#### 介绍
用于地震学处理分析的开源工具，目前包括：
震相拾取、极化、频散提取

#### 软件架构
软件完全基于Python构建
每个项目均是独立的，依赖于：
- obspy：数据读取
- PyTorch：深度学习库
- OpenCV：图像处理


#### 安装教程

1.  建议安装Anaconda最新版本
2.  其他库可以使用pip进行安装
3.  深度学习库建议使用conda

基础环境安装过程参考文章
